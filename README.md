# ⚡ VoltStore — Laravel E-Commerce Platform

Welcome to **VoltStore**, a modern and scalable e-commerce platform built using the powerful **Laravel** framework. This project showcases a complete online shopping experience with a clean UI, robust backend, and efficient performance.

---

## 🚀 Features

* 🛒 Fully functional shopping cart system
* 🔍 Product browsing & search
* 🧾 Order management
* 👤 User authentication & authorization
* 💳 Checkout system (extendable for payment gateways)
* 📦 Admin panel for product & order control
* ⚡ Fast and responsive UI

## ⚙️ Installation Guide

Follow these steps to set up the project locally:

```bash
# Clone the repository
git clone https://github.com/abdurrehman214/volt-store-an-e-commerce-store-built-using-laravel.git

# Navigate into the project directory
cd volt-store-an-e-commerce-store-built-using-laravel

# Install dependencies
composer install

# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Configure your database in .env

# Run migrations
php artisan migrate

# Start the development server
php artisan serve
```
